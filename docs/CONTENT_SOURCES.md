## Adding New Content

When contributing new content to ChaosLingua:

1. **Verify licensing** - Ensure content is public domain, Creative Commons, or you have permission
2. **Provide attribution** - Add source information to this document
3. **Respect terms** - Follow any platform-specific embedding requirements
4. **Document sources** - Include links to original content where applicable

## Questions or Concerns

If you have questions about content licensing or believe content has been incorrectly attributed, please open an issue on the GitHub repository.
